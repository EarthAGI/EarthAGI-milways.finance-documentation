# Resetting Password
How to reset your Milways.Finance password

<Info>
 This guide explains how to reset your password for Milways Finance. It's a step-by-step tutorial that walks you through the entire password recovery process, from clicking the "Forgot Password" button to successfully setting a new password.
</Info>

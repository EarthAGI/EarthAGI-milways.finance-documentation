<Info>
Milways.Finance 지갑에 자금을 입금하는 것은 신용카드 한도 충전이나 송금과 같은 플랫폼의 다양한 기능을 사용하기 위한 기초입니다. 이 가이드는 암호화폐를 지갑에 안전하게 입금하는 방법을 보여줍니다.
</Info>

## 시작하기

![지갑 충전 1](https://www.gitbook.com/cdn-cgi/image/dpr=2,width=760,onerror=redirect,format=auto/https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FYsrj5O8moOrzPyGksbuI%2Fblobs%2FjeIv7NmRcAmY1kjJtplH%2F4%2520(15).png&width=400&dpr=3&quality=100&sign=26fb37e8&sv=2)

1. 💼 지갑 메인 페이지에 액세스하여 **"담보"** 버튼을 탭합니다.
2. 🔄 **"입금하기"**를 클릭하여 계속합니다.

## 네트워크 선택 및 주소 받기

![지갑 충전 2](https://www.gitbook.com/cdn-cgi/image/dpr=2,width=760,onerror=redirect,format=auto/https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FYsrj5O8moOrzPyGksbuI%2Fblobs%2FGE49bdu4klwGZyvXYfAU%2F33.png&width=400&dpr=3&quality=100&sign=8d89ece&sv=2)

3. 🌐 **네트워크 선택**: QR 코드 아래에서 입금에 사용할 블록체인 네트워크를 선택할 수 있습니다. Milways.Finance는 Tron(TRC20), Ethereum(ERC20), Solana(SOL)를 포함한 여러 네트워크를 지원합니다. 자금을 보내는 외부 지갑이나 거래소와 일치하는 네트워크를 선택하십시오.
4. 🔢 **입금 주소 받기**: 네트워크(예: Tron 네트워크)를 선택하면 시스템에 해당 입금 주소(QR 코드 및 텍스트 형식 모두)가 표시됩니다. 주소 아래의 **"복사"** 버튼을 클릭하여 클립보드에 복사합니다.

<Warning>
**중요 사항:**
* 선택한 네트워크가 자금을 보내는 데 사용하는 네트워크와 **정확히 일치**하는지 확인하십시오. 잘못된 네트워크 주소로 자금을 보내면 **영구적으로 손실**될 수 있습니다.
* 각 네트워크에는 고유한 입금 주소가 있습니다. 복사한 주소를 신중하게 확인하십시오.
* 복사한 주소를 자금을 보내는 지갑이나 거래소의 출금 페이지에 붙여넣습니다.
* 입금에는 일반적으로 네트워크 확인 시간이 걸립니다. 잠시 기다려 주십시오.
</Warning>

<Tip>대규모 입금을 하기 전에 모든 것이 올바르게 작동하는지 확인하기 위해 먼저 소액 테스트 송금을 하는 것이 좋습니다.</Tip>

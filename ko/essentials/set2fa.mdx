<Info>
이중 인증(2FA)은 Milways.Finance 계정을 보호하기 위한 중요한 보안 조치입니다. 2FA를 활성화하면 로그인하거나 민감한 작업을 수행할 때 비밀번호 외에도 인증 앱(예: Google Authenticator)의 동적 확인 코드를 입력해야 하므로 계정 보안이 크게 강화됩니다. 모든 사용자에게 2FA를 활성화하는 것을 강력히 권장합니다.
</Info>

## 2FA 설정으로 이동

![2FA 설정 1](https://www.gitbook.com/cdn-cgi/image/dpr=2,width=760,onerror=redirect,format=auto/https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FYsrj5O8moOrzPyGksbuI%2Fblobs%2FaQXKzhofhnVg0PqMZRa7%2F45.png&width=400&dpr=3&quality=100&sign=369bbab9&sv=2)

1. 🔧 Milways.Finance 앱을 열고 오른쪽 상단의 설정 아이콘을 탭합니다.
2. 🔒 설정 페이지로 이동하여 **"2FA 재설정"** 옵션을 선택합니다.

![2FA 설정 2](https://www.gitbook.com/cdn-cgi/image/dpr=2,width=760,onerror=redirect,format=auto/https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FYsrj5O8moOrzPyGksbuI%2Fblobs%2F4R8il3vqbNzpWFWhdUqO%2F46.png&width=400&dpr=3&quality=100&sign=f3346cfa&sv=2)

3. 📋 QR 코드 페이지에서 **"2FA 복사"**를 탭하여 확인 키를 복사합니다.

<Warning>
**매우 중요: 이 키를 안전한 곳에 백업해 두십시오!**
* 휴대전화를 분실하거나 교체하는 경우 이 키가 2FA 액세스를 복구하는 유일한 방법입니다.
* 키를 손으로 적어 안전한 곳에 보관하거나 안전한 비밀번호 관리자에 저장하십시오.
* 키를 스크린샷이나 일반 텍스트 파일로 연결된 기기에 저장하지 마십시오.
* 백업 키가 없는 경우 인증 앱에 액세스할 수 없게 되면 Milways.Finance 계정에 영구적으로 액세스하지 못할 수 있습니다!
</Warning>

![2FA 설정 3](https://www.gitbook.com/cdn-cgi/image/dpr=2,width=760,onerror=redirect,format=auto/https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FYsrj5O8moOrzPyGksbuI%2Fblobs%2Fy6MycACUqPT5QKQ6bKys%2F47.png&width=400&dpr=3&quality=100&sign=c37b0670&sv=2)

4. 📱 Google Authenticator 또는 인증 앱을 열고 **"코드 추가"**를 탭합니다.
5. 📝 계정 이름을 입력합니다(예: Milways.Finance).
6. 📝 복사한 키를 "키" 필드에 붙여넣습니다.

![2FA 설정 4](https://www.gitbook.com/cdn-cgi/image/dpr=2,width=760,onerror=redirect,format=auto/https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FYsrj5O8moOrzPyGksbuI%2Fblobs%2FF9ZdX0XIUj8apm7bytIy%2F48.png&width=400&dpr=3&quality=100&sign=3b3f84eb&sv=2)

7. 🔢 **확인 코드 생성**: Google Authenticator가 6자리 확인 코드(예: 100 075)를 생성합니다.

![2FA 설정 5](https://www.gitbook.com/cdn-cgi/image/dpr=2,width=760,onerror=redirect,format=auto/https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FYsrj5O8moOrzPyGksbuI%2Fblobs%2FsWdqBNOoOfywmfYXxrxB%2F49.png&width=400&dpr=3&quality=100&sign=268ae856&sv=2)

8. 📝 Milways.Finance 앱으로 돌아가 **"붙여넣기"**를 사용하여 6자리 코드를 입력합니다.

![2FA 설정 6](https://www.gitbook.com/cdn-cgi/image/dpr=2,width=760,onerror=redirect,format=auto/https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FYsrj5O8moOrzPyGksbuI%2Fblobs%2F3TX6DOCta4s0Ih7okOcR%2F50.png&width=400&dpr=3&quality=100&sign=33f70379&sv=2)

9. 🟢 **2FA 활성화**: **"2FA 활성화"** 버튼을 탭합니다. 인터페이스 하단에 녹색 "2FA 설정 성공" 표시가 나타납니다.
10. 📧 **확인 이메일**: Milways.Finance로부터 계정에서 2FA가 성공적으로 활성화되었음을 확인하는 이메일을 받게 됩니다.

축하합니다! Milways.Finance 계정에 보안 계층을 추가했습니다. 백업 키를 안전하게 보관하는 것을 잊지 마십시오.

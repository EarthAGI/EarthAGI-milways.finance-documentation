<Info>
Depositing funds into your Milways.Finance wallet is fundamental to using the platform's various features, such as recharging credit card limits or making transfers. This guide will show you how to safely deposit cryptocurrency into your wallet.
</Info>

## Choose Network and Get Address

![Recharging Wallet 2](https://www.gitbook.com/cdn-cgi/image/dpr=2,width=760,onerror=redirect,format=auto/https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FYsrj5O8moOrzPyGksbuI%2Fblobs%2FGE49bdu4klwGZyvXYfAU%2F33.png&width=400&dpr=3&quality=100&sign=8d89ece&sv=2)

3. ğŸŒ **Choose Network**: Below the QR code, you can select which blockchain network you want to use for the deposit. Milways.Finance supports multiple networks, including Tron (TRC20), Ethereum (ERC20), and Solana (SOL). Make sure to select a network that matches the external wallet or exchange you're sending funds from.
4. ğŸ”¢ **Get Deposit Address**: After selecting a network (e.g., Tron network), the system will display the corresponding deposit address (in both QR code and text format). Click the **"Copy"** button below the address to copy it to your clipboard.

<Warning>
**Important Notes:**
* Make sure the network you choose **exactly matches** the network you're using to send funds. Sending funds to an incorrect network address may result in **permanent loss**.
* Each network has its own deposit address. Please carefully verify the address you've copied.
* Paste the copied address into the withdrawal page of the wallet or exchange you're transferring funds from.
* Deposits typically require some network confirmation time, please be patient.
</Warning>

<Tip>It's recommended to first make a small test transfer to ensure everything works correctly before making larger deposits.</Tip>
